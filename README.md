🔐 Task-4: Advanced Encryption Tool

📌 What is Encryption?

Encryption means converting readable data into secret code so that unauthorized people cannot read it.

Example:

Original text:

Hello

Encrypted text:

a8f93bd12c

Only someone with the correct key can decrypt it.

🎯 Objective of Task-4

Create a Python tool that can encrypt and decrypt files using AES-256 encryption.

🔑 What is AES-256?

AES means Advanced Encryption Standard.
It is one of the most secure encryption algorithms in the world.
AES-256 uses a 256-bit key to protect data.

Real systems using AES:

Banking systems

VPNs

Password managers

Cloud storage

Example:

BitLocker

VeraCrypt

⚙️ How the Encryption Tool Works

Step 1

User selects a file.

Example:

document.txt

Step 2

Program encrypts the file.

document.txt → encrypted_file.bin

Now the file cannot be read.

Step 3

Decryption

User enters the same key.

encrypted_file.bin → document.txt

The original file is restored.
